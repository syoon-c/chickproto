# 핵심 플레이 연구

`홍보 -> 손님 등장 -> 주문 받기 -> 요리 하기 -> 음식 주기 -> 식사 -> 퇴장`만 남긴 최소 프로토타입입니다.

## 남겨둔 것

- 병아리/시설/음식 리소스
- 홍보 버튼
- 손님 1명 등장
- 주문받기, 요리하기, 음식 주기 버튼
- 식사 후 퇴장까지의 단계별 상태 전이
- 간단한 로그와 카운트
- 브라우저 `localStorage` 저장 및 리셋

## 제거한 것

- 만족도
- 승급
- 공부
- 메뉴 연구/강화
- 확장
- 외출
- 농장
- 면접
- 직원
- SNS
- 도감
- 기타 데이터 브리지와 산출물

## 실행

아무 정적 서버로 열면 됩니다.

```bash
cd "C:\Users\Soyoon Bang\Desktop\핵심플레이 연구"
python -m http.server 4173
```

브라우저에서 `http://127.0.0.1:4173` 접속 후 `홍보하기 -> 주문받기 -> 요리하기 -> 음식 주기` 순서로 확인하면 됩니다.
